# PacketAnalyzer
グラフィカルなインターフェースでパケットの解析を可能にします

## 特徴
1. Spigotのバージョンに関わらず動作します
2. Windows/Mac/Linuxに対応します(要GUI環境)
3. パケットのデータを詳細に確認できます
4. 時系列順にパケットを確認できます
